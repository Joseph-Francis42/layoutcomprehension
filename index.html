04 // CSS Floats & Clearfix

Flow Text Wrapping

Floats are ideal for wrapping document content naturally around inline containers like sidebars or images.

Floated Aside

This box is floated left with a right margin. Notice how the paragraph content wraps smoothly around its borders.

float: left;

Floats remove an element from normal flow, placing it along the left or right edge of its parent container. The inline elements in the document continue to flow around it.

When elements are floated, parent containers collapse if they contain only floated children, because floated nodes do not occupy heights in the normal page flow. This requires a clearfix. In this example, the wrapper container uses the classic ::after pseudo-element rule to clear float nodes without modifying HTML structures.

While CSS Grid and Flexbox are preferred for overall page structure, floats remain the primary tool for wrapping editorial text smoothly around side graphics and photos.
